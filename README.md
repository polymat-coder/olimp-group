# Олимп Групп

Статическая копия сайта компании «Олимп Групп», подготовленная для GitHub Pages.

## Состав

- `index.html` и остальные HTML-файлы — страницы сайта.
- `_next/static/` — клиентские стили, JavaScript и шрифты Next.js.
- `avto*.MP4` и `avto*.jpg` — видеоматериалы и превью.
- `dogovor.pdf` — договор в PDF.
- `.github/workflows/pages.yml` — автоматическая публикация на GitHub Pages.

## Локальный запуск

```bash
node scripts/serve.mjs
```

После запуска сайт доступен по адресу `http://127.0.0.1:8765/`.

## Подготовка путей

Если исходные экспортированные файлы обновлены, выполните:

```bash
node scripts/prepare-pages.mjs
```

Скрипт приводит пути Next.js и внутреннюю навигацию к формату, совместимому с
размещением проекта в подпапке GitHub Pages.
